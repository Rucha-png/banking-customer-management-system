# Banking Customer Management System

Spring Boot CRUD application using Spring Data JPA, Hibernate and MySQL.

## Technologies
- Java 17
- Spring Boot 3.5
- Spring Data JPA
- Hibernate
- MySQL
- Maven

## Setup

Configure the following environment variables before running:

```bash
DB_USERNAME=your_mysql_username
DB_PASSWORD=your_mysql_password
```

Database:

```sql
CREATE DATABASE banking_db;
```

Run:

```bash
mvn spring-boot:run
```
